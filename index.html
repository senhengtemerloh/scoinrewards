How to Use

  • Enter S-Coin values in RM (e.g., 100, 200, 500) in the input fields. You can enter decimal values.
  • Click "**Generate POP Cards**" to create your Proof of Purchase cards.
  • Use "**Clear All**" to reset all inputs and generated cards.
  • Click "**Print (A4 optimized)**" to print your generated POP cards. The layout and colors will be consistent with the on-screen display for optimal printing on A4 sheets.
  • Each card will show suggested products that fit within the S-Coin value, prioritizing the best value match.
  • Product data is loaded from the `data.xlsx` file in the same directory. If not found, sample data will be used.
